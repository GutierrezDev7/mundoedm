# Fontes do site

Para usar **Gotham** no site, coloque aqui os arquivos da família Gotham:

- `Gotham-Book.woff2` (e/ou `Gotham-Book.woff`) — peso 400
- `Gotham-Medium.woff2` (e/ou `Gotham-Medium.woff`) — peso 500
- `Gotham-Bold.woff2` (e/ou `Gotham-Bold.woff`) — peso 700

Gotham é uma fonte comercial. Se os arquivos não estiverem nesta pasta, o site usará a fonte de fallback (Geist).
